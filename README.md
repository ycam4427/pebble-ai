### Install
1. Install [Ollama](https://ollama.com) and run `ollama pull llama3.2`
2. Download & run **`Pebble_0.9.0_x64-setup.exe`** below

> Windows may show "Windows protected your PC" (it's an unsigned app) — click **More info → Run anyway**. It's safe & open source.

Pebble is a small local AI — he can make mistakes, so review his suggestions. Be gentle 🤍
